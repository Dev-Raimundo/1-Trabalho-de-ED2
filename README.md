# Regressão Logística: Vetorização vs. Implementação Iterativa
**Projeto da disciplina Estrutura de Dados II - UNIFESSPA**

Este repositório contém o estudo comparativo de desempenho entre a implementação de Regressão Logística utilizando laços `for` (iterativo) e operações matriciais (vetorizado com NumPy).

## 🚀 Resultados Principais
- **Dataset:** Titanic (Kaggle)
- **Speedup Alcançado:** 89.74x
- **Convergência Matemática:** Ambas as versões atingiram o custo exato de 0.462209.

## 🛠️ Tecnologias Utilizadas
- Python 3.14.3
- NumPy (Computação Científica)
- Matplotlib (Visualização de Dados)
- Pandas (Tratamento de Dados)

## 📁 Estrutura do Repositório
- `src/`: Códigos-fonte das implementações.
- `docs/`: Relatório científico completo em PDF.
- `requirements.txt`: Dependências do projeto.
